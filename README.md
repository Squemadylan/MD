# MD阅读器 v1.4.2

一款支持多标签页的 Windows Markdown 阅读器

## ✨ 功能特性

### 🗂️ 多标签页管理
- 同时打开多个 Markdown 文件，标签页可拖拽排序
- 顶部标签指示线设计，清晰展示当前激活标签
- 单窗口内流畅切换，高效管理多文档

### 🎨 现代化 UI 设计
- **双主题模式**：明亮模式 & 暗黑模式，支持手动切换或自动同步 Windows 系统设置
- **渐变按钮**：采用现代渐变设计风格
- **圆角设计**：全局圆角元素，视觉更加柔和舒适
- **顶部标签指示线**：标签页顶部彩色指示线，状态一目了然

### 📝 智能编码与渲染
- **自动编码检测**：智能识别 UTF-8 / GBK / GB2312 等编码格式，中文显示无乱码
- **完整 Markdown 渲染**：支持标题、列表、表格、代码块、任务列表等全部 Markdown 格式
- **代码高亮**：支持多种编程语言的语法高亮

### 🔄 智能文件管理
- **拖拽打开**：直接拖拽 `.md` 文件或文件夹到窗口即可打开
- **自动恢复**：重启后自动打开上次编辑的文件和目录，无缝续接工作
- **最近文件记录**：快速访问最近打开的文档

### 🛡️ 文件保护机制
- **大文件警告**：文件超过 10MB 时自动提示，避免加载卡顿
- **空文件提示**：检测空文件并友好提示
- **异常读取保护**：文件读取异常时优雅处理，不崩溃

## 🚀 使用方法

### 启动程序
```bash
pip install PySide6
python md_reader.py
```

或双击 `启动MD阅读器.bat`

### 界面操作

| 按钮 | 功能 |
|------|------|
| 📂 打开 | 打开 Markdown 文件（可多选） |
| 📂 打开文件夹 | 打开整个文件夹批量加载 |
| ➕ 新标签 | 创建空白标签页 |
| 🔄 刷新 | 重新加载当前标签页的文件 |
| ❌ 关闭 | 关闭当前标签页 |
| ☀️/🌙 | 切换明亮/暗黑主题（或跟随系统） |
