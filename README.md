# MD阅读器 v1.3.1

一款支持多标签页的 Windows Markdown 阅读器，v1.3.1 修复了 JS 加载时序问题。

## ✨ 功能特性

- 🗂️ **多标签页**：同时打开多个 Markdown 文件，标签页可拖拽排序
- 🎨 **双主题模式**：明亮模式 & 暗黑模式
- 🔄 **一键切换**：点击 ☀️/🌙 图标按钮即时切换主题
- 📝 **完整渲染**：支持标题、列表、表格、代码块等全部 Markdown 格式
- 🖱️ **拖拽打开**：直接拖拽 .md 文件到窗口即可打开
- 🐛 **时序修复**：v1.3.1 修复了偶尔不显示内容的问题

## 🚀 使用方法

### 启动程序
```bash
pip install PySide6
python md_reader.py
```

或双击 `启动MD阅读器.bat`

### 界面操作

| 按钮 | 功能 |
|------|------|
| 📂 打开 | 打开 Markdown 文件（可多选） |
| ➕ 新标签 | 创建空白标签页 |
| 🔄 刷新 | 重新加载当前标签页的文件 |
| ❌ 关闭 | 关闭当前标签页 |
| ☀️/🌙 | 切换明亮/暗黑主题 |

## 📝 修复说明

### v1.3.1 修复内容

**问题**：打开文件时偶尔不显示内容，控制台报错 `renderMarkdown is not defined`

**原因**：WebView 页面加载和 JavaScript 代码执行存在竞态条件，`marked.js` 库未完全加载时调用渲染函数

**解决方案**：
1. 使用 `window.addEventListener('load')` 确保页面完全加载
2. 添加 `window.pendingContent` 队列暂存待渲染内容
3. 渲染前检查 `marked` 和 `renderMarkdown` 是否已定义
4. 添加延迟重试机制（200ms 间隔最多重试）
